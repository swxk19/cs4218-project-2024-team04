1) To run the performance test, ensure that Ultimate Thread Group plugin is installed. Steps can be found here https://jmeter-plugins.org/install/Install/
2) Ensure that MongoDB has the users accounts filled with "cs4218-test.users.json"
3) Open the "Spike Test.jmx" with JMeter
4) Replace the Path in the "Extract Saved Users Details" to the path of "users.csv"
5) Run test
6) "data_final.xlsx" contains the processed data after running the test
